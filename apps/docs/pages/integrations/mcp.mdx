## Automatic Setup

<Tip>Use the kyoto cli to automatically setup</Tip>

```sh
kyoto setup
```

Follow the instructions to configure the MCP service which includes:

- Adding Kyoto to `.cursor/mcp.json`
- Creating a `.cursor/rules/kyoto.mdc` for agents to follow

## Manual Setup

First you need to register the MCP service, then give instructions to your agents to use them.

**Register Kyoto's MCP**

<CodeGroup>

```json Cursor
// .cursor/mcp.json
{
  "services": {
    "Kyoto": {
      "command": "npx",
      "args": ["kyoyo", "mcp"]
  }
}
```
