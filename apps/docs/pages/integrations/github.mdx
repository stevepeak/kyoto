## Automatic Setup

<Tip>Use the Kyoto CLI to automatically set this up</Tip>

```sh
kyoto setup github
```

Follow the instructions to configure the GitHub Actions workflow, which includes:

- Creating a GitHub Action at `.github/workflows/kyoto.yml`

## Manual Setup

Create a GitHub Actions workflow at `.github/workflows/kyoto.yml`

```yml .github/workflows/kyoto.yml
name: Kyoto

on:
  push:
    branches:
      - main
  pull_request:

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: '22'
      - run: npx @usekyoto/cli vibe check
```
