## Install

<CodeGroup>

```sh bun
bun install -g kyoto
```

```sh npm
npm install -g kyoto
```

```sh pnpm
pnpm install -g kyoto
```

</CodeGroup>

## Setup

Once the kyoto cli is installed, navigate to your
git repository and setup kyoto via:

```sh shell
kyoto setup
```

This command will guide you through:

1. Adding `.kyoto` to your `.gitignore`
2. Choosing your AI provider
3. Setting up the [Kyoto MCP](/pages/integrations/mcp)
4. Configuring a new [GitHub Action for Kyoto](/pages/integrations/github)
